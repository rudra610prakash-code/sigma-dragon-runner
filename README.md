## ✨ Features

- 🎮 **Intuitive Controls** - Simple keyboard controls for smooth gameplay
- 🐲 **Dynamic Obstacles** - Dragon enemies with increasing difficulty
- 💥 **Collision Detection** - Accurate hit detection system
- 📊 **Score System** - Track your progress and beat high scores
- 🔊 **Sound Effects** - Interactive audio feedback during gameplay
- ✨ **Smooth Animations** - Fluid sprites and transitions
- 📱 **Responsive Design** - Works on desktop browsers

## 🎮 How to Play

| Key | Action |
|-----|--------|
| **⬆️ Arrow Up** | Jump |
| **⬅️ Arrow Left** | Move Left |
| **➡️ Arrow Right** | Move Right |

**Objective:** Avoid the incoming dragon obstacles and survive as long as possible to maximize your score!
